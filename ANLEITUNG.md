📅 Foto-Datierung – Anleitung
Voraussetzungen
Python 3.8+ → https://python.org/downloads (Bei der Installation ✅ "Add Python to PATH" aktivieren!)
Starten
Doppelklick auf: "Foto-Datierung starten.bat"
Das Skript installiert beim ersten Start automatisch tkinterdnd2 (wird für Drag & Drop benötigt).
So funktioniert es
Fotos hineinziehen
Fotos aus dem Explorer direkt in den blauen Bereich ziehen
Oder auf den Bereich klicken → Datei-Dialog öffnet sich
Beliebig viele Fotos, beliebige Reihenfolge
Reihenfolge anpassen
Zeilen in der Liste per Drag & Drop sortieren (am ⠿ Symbol ziehen)
Mit ✕ einzelne Fotos entfernen
Datum & Uhrzeit eingeben
Format: TT.MM.JJJJ und HH:MM
Beispiel: 15.07.2024 und 14:30
"✅ Datum anwenden" klicken
Alle Fotos bekommen exakt dieses Datum
Windows-Dateidatum (Erstell- + Änderungsdatum) wird geändert
Originaldateien werden direkt überschrieben
Reset – Liste leeren und neu starten
Unterstützte Formate
JPG, JPEG, PNG, GIF, BMP, TIFF, WEBP, HEIC, HEIF, RAW, CR2, NEF, ARW, MP4, MOV, AVI, MKV
Hinweise
Die App ändert nur das Windows-Dateidatum, nicht EXIF-Metadaten
Windows Explorer sortiert Fotos nach diesem Datum
Keine Internetverbindung nötig
